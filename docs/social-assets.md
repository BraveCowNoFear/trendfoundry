# Social Assets

Use this when refreshing link previews for outreach, checkout pages, or social posts.

## Command

```bash
npm run social
npm run build
```

Generated files:

- `site/og-image.png`
- `dist/social/og-card.html`

`site/og-image.png` is committed because GitHub Pages deploys static files only and should not depend on a local browser renderer.

## Visual Rules

- Keep the image factual: source-backed, creator intelligence, price ladder, no growth promises.
- Do not use fake charts, inflated metrics, or platform-growth claims.
- Match the main site: white background, dark ink, restrained accent colors, information-carrying source bar.
