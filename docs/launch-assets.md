# Launch Assets

Use this to prepare manual launch posts and warm outreach copy without posting or sending anything automatically.

## Command

```bash
npm run launch-assets
```

Generated local outputs:

- `dist/launch-assets/launch-posts.md`
- `dist/launch-assets/launch-posts.csv`
- `docs/launch-posts.md`

## Channels

The generated pack includes draft copy for:

- X / short launch thread
- LinkedIn
- Bilibili dynamic
- GitHub pinned comment or repository update
- warm intro email

## Safety

- Review every draft before posting or sending.
- Do not bulk-send identical messages.
- Do not promise views, revenue, subscribers, or platform growth.
- Do not ask buyers to post payment card details or private IDs in public replies.
