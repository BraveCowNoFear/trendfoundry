# TrendFoundry 工作区

## 负责人决策

- 项目负责人：Codex，按勇敢牛牛授权自动推进。
- 当前方向：面向 B 站/YouTube AI 与开发者创作者的趋势情报包。
- 选择理由：2026 年创作者仍在购买省时间工作流，但平台正在打击低质量 AI 内容；有来源、有复现实测、有创作结构的情报包比批量内容更安全、更可卖。
- 当前 MVP：自动采集 GitHub、YouTube、Bilibili、Hacker News、arXiv 公开趋势信号，生成带评分的创作者选题包、日报 Markdown 和可筛选静态情报台。

## 外部证据样本

- Creator economy 2026 报告与行业文章普遍提到 AI 工具进入创作者生产流程，创作者自有产品与订阅是更稳的收入路径。
- YouTube 2026 相关报道强调会支持 AI 创作工具，但会继续压低质量、重复、欺骗性 AI 内容。
- GitHub topic `video-generation` 已有大量公开仓库，AI agent / video generation / creator workflow 是持续更新的技术内容池。
- Hacker News、arXiv、GitHub、YouTube 搜索页和 Bilibili 公共搜索接口都能提供无需账号/钱包前置依赖的公开趋势信号；其中 GitHub/Bilibili 可能限流或 412，需缓存兜底。

## 产品定义

TrendFoundry 每天/每周输出：

- 12 个有来源链接的创作者机会。
- 每个机会包含评分、受众、B 站标题、YouTube 标题、开场 hook、视频结构、缩略图提示词。
- 最高分机会可扩展成完整脚本、素材清单和演示步骤。

## 当前文件

- `package.json`：本地命令。
- `scripts/collect.mjs`：公开数据采集和机会评分。
- `scripts/build.mjs`：生成 Markdown brief 和静态站点。
- `scripts/server.mjs`：本地预览服务器。
- `scripts/run_daily.ps1`：Windows 日更脚本。
- `scripts/export_pack.mjs`：生成买家可收取的样品包、机会 CSV、外联 CSV 和外联板。
- `data/seed_sources.json`：初始查询主题。
- `data/latest.json`：最近一次采集结果，含 `errors` 与 cached fallback 标记。
- `docs/daily-brief.md`：当前可售卖样品。
- `docs/ready-to-record-script.md`：第一条可录视频脚本。
- `docs/sales-page-copy.md`：支付页/销售页文案。
- `docs/launch-plan.md`：首发计划、客户画像、触达话术。
- `docs/outreach-playbook.md`：客户触达话术、分层和履约 SOP。
- `docs/outreach-board.md`：`npm run export` 生成的 20 人首发外联板。
- `docs/payment-platform-fields.md`：支付平台上架字段。
- `site/index.html`：展示页。
- `site/app.js`：站内搜索与来源筛选交互。
- `site/*.md`：供展示页按钮直接打开的样品交付物镜像。
- `dist/trendfoundry-sample-pack/`：`npm run export` 生成的可发送样品包，不纳入 Git。
- `记忆系统.md`：项目长期记忆与决策日志。
